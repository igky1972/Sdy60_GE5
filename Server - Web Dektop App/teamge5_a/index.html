Αρχική

Το WalkPathsOnMap είναι μια εφαρμογή Web που δημιουργήθηκε στα πλαίσια της 5ς ΓΕ της ΘΕ ΣΔΥ 60 του ΜΠΣ ΣΔΥ του ΕΑΠ από την ομάδα εργασίας TeamGE5_A που αποτελείται από τους:

  • Ηρακλή Γκυρτή
  • Ιωάννη Κούστα
  • Γεώργιο Λιόκαλο
    •
    Για τη δημιουργία της, στηριχτήκαμε στη Διπλωματική Εργασία του Μεταπτυχιακού Φοιτητή του ΕΑΠ κ. Ιπποκράτη Καπενακάκη.

    Ο τελικός στόχος της εφαρμογής είναι να δημιουργηθεί ένας χάρτης για πεζοδιαδρομές στο Ρέθυμνο.

    Υπάρχει και αντίστοιχη εφαρμογή για συσκευές Android η οποία είναι διαθέσιμη στους χρήστες από το σύνδεσμο που ακολουθεί:
